# CYFX — A 股价值投资日报

## 功能概述

自动生成高质量 A 股价值投资日报，包含以下核心板块：

1. **市场概览**：指数表现、领涨领跌板块、宏观与海外市场、市场情绪判断
2. **热点板块分析**：大模型/Agent、AI 算力/半导体、新能源/固态电池、具身智能/机器人、自动驾驶、AI 医疗、航空航天/卫星、券商/金融科技
3. **产业链六层框架**：电力层、机房层、存储层、芯片层、网络层、大模型/Agent 层
4. **资金面与板块热度**：全市场成交额、机构动向、目标价调整
5. **本周重要事件日历**：日期、事件、影响板块、重要性评级
6. **风险提示**：短期风险（1-4 周）+ 中期风险（1-3 个月）
7. **投资策略建议**：保守型、稳健型、进取型三种方案

## 无需配置

此技能开箱即用，无需 API Key。

## 命令行调用

```bash
# 生成投资日报并发送邮件
python3 skills/CYFX/scripts/generate_daily_report.py

# 指定日期
python3 skills/CYFX/scripts/generate_daily_report.py --date 2026-04-29

# 仅生成不发送
python3 skills/CYFX/scripts/generate_daily_report.py --dry-run
```

## 撰写要求

- 内容需结合最新资讯和市场趋势，体现价值投资理念，避免投机性分析
- 语言专业、简洁，数据引用准确，逻辑清晰
- 每个板块需有明确的标题，结构层次分明
- 确保信息真实可靠，不传播虚假或误导性内容，标注来源
- 使用 HTML 格式，专业配色（红涨绿跌），表格清晰，卡片式布局

## 报告结构

### 📊 一、今日市场概览
- A 股市场核心观点（融资余额、市场资金流向）
- 宏观与海外市场（汇率与流动性、美股与全球、地缘政治）
- 市场情绪判断（🟢偏多 / 🟡中性 / 🔴偏空 + 3-4 条核心逻辑）

### 🔥 二、热点板块分析
大模型和 Agent、AI 算力/半导体、新能源/固态电池、具身智能/机器人、自动驾驶、AI 医疗和 AI 创新药、航空航天和卫星通信、券商/金融科技

每条要闻必须包含：
- 事件（1-2 句话概括）
- 催化（1-3 个具体催化剂）
- 投资启示（对 A 股/港股/美股的具体影响）
- 确定性评级（🟢高 / 🟡中 / 🔴低）
- 来源（媒体+日期）

### 🏗️ 三、产业链动态分层分析
按以下 6 层分析，每层包含：
1. 电力层：算电协同、液冷
2. 机房层：超节点，算力租赁、AIDC 建设
3. 存储层：DRAM/NAND/HBM价格+国产标的
4. 芯片层：英伟达/英特尔/寒武纪/海光/芯原等
5. 网络层：光模块、交换机、网络安全
6. 大模型和 Agent 层：最新大模型和 Agent 动态关联的投资机会

每层必须包含：
- 标的表格（标的 | 最新动态 | 投资看点 | 确定性）
- 核心逻辑（2-3 句话）
- 投资策略（1 句话）

### 💰 四、资金面与板块热度
- 全市场成交额
- 机构动向（目标价调整、评级变化）

### 📅 五、本周重要事件日历
- 日期 | 事件 | 影响板块 | 重要性（⭐）
- 特别标注重大催化（如英伟达财报、人工智能+和新型电网、算力网等六大网新基建建设）

### 六、风险提示
- 按优先级排序的 7 大风险点
- 短期风险（未来 1-4 周）
- 中期风险（未来 1-3 个月）

### 七、投资策略建议
- 保守型、稳健型、进取型三种方案
